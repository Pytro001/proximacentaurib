# PEAK M Industries Website

A modern, SpaceX-inspired website for PEAK M Industries - Upgrading Human Capabilities.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Dark, modern design inspired by SpaceX
- Responsive layout for all devices
- Hero section with rotating background images
- Smooth animations and transitions
- Clean navigation and footer

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- CSS Modules
